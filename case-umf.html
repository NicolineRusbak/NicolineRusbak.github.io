Ungdomsringens Musikfestival

Intro til opgaven

Målet for projektet var at lave et responsivt website for Ungdomsringens Musikfestival. Formål med siden er at samle alle informationer om festivallen på et website. Siden skulle være langtidsholdbar, dvs. det skulle være nemt at skifte indholdet. Den skulle primært henvende sig mod nye og tilbagevendende deltagere og sekundært mod festivalens øvrige publikum.

Forarbejde

Vi startede som altid med at samle en masse informationer om kunden og brugerne. Det gjorde vi ved hjælp af brugerinterviews med personer i målgruppen. Derefter lavede vi et content inventory og benchmarking. Dette lagde grundlag for kravspecifikationen, idégenereringen og valget af indhold.

Realisering

Produktet er realiseret med HTML, CSS og JavaScript. Siden er lavet som desktop-first med brug af grid, der gør det nemmere at nedskalere det senere. Der er ligeledes arbejdet med teknisk SEO, i form af semantisk struktur og meta tekst.

Ansvarsområder:

  • Benchmarking
  • Lave wireframe og mockup
  • Udvikling af HTML, CSS og JavaScript

Værktøjer og teknologier:

  • Adobe XD
  • HTML
  • CSS
  • JavaScript
